# Wealthmap
By: Trojan

> A mobile-first financial wellness platform empowering African youth to budget, save, invest, and protect their financial future.

--- 

## Overview

WealthMap is a financial wellness platform built for African youth aged 18–25. It combines AI-powered guidance with practical tools for budgeting, savings tracking, investment roadmaps, fraud awareness, and financial education — all in one accessible interface.

---

## Features 

- **Budgeting** — Track income and expenses with smart categorization
- **Savings Tracker** — Set and monitor savings goals
- **Investment Roadmaps** — Beginner-friendly investment pathways tailored to the African market
- **Fraud Awareness** — Alerts and education on common financial scams
- **AI Financial Assistant** — Powered by the Claude API for personalized guidance
- **Financial Education** — Bite-sized lessons and resources

## Getting Started

### Prerequisites

```bash
node >= 18.x
postgresql >= 15.x
```

### Installation

```bash
# Clone the repository
git clone https://github.com/EricChuwa/wealthmap.git
cd wealthmap

# Install dependencies
npm install

# Set up environment variables
cp .env.example .env

# Run the development server
npm run dev
```
